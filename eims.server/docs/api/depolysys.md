# Depolysys API

`DepolysysController` — 部署系統 CRUD 與條件查詢。

| Method | Path | 說明 |
|---|---|---|
| GET | `/depolysyss` | 清單 |
| POST | `/depolysyss` | 新增 |
| PUT | `/depolysyss` | 修改 |
| GET | `/depolysyss/{deploySysCd}` | 取單筆 |
| DELETE | `/depolysyss/{deploySysCd}` | 刪除 |
| POST | `/depolysyss/getlist` | 條件查詢清單 |

詳見 [`openapi.yaml`](./openapi.yaml) tag `Depolysys`。
